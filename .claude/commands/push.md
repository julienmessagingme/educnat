# Push - Commiter et pousser sur GitHub

Commite tous les changements et pousse sur GitHub.

## Étapes à exécuter

1. Lancer `git status` pour voir les fichiers modifiés
2. Lancer `git diff --stat` pour résumer les changements
3. Ajouter les fichiers modifiés avec `git add` (fichiers spécifiques, pas -A)
4. Créer un commit avec un message descriptif en français
5. Pousser avec `git push`
6. Afficher l'URL du commit sur GitHub
